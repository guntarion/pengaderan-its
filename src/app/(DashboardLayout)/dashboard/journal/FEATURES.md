# Fitur Jurnal Mingguan — M04

## Deskripsi

Modul Jurnal Mingguan menyediakan ruang refleksi terstruktur bagi MABA
untuk mendokumentasikan pengalaman belajar setiap minggu menggunakan
kerangka What Happened / So What / Now What.

## Fitur Pengguna

### Daftar Jurnal
Halaman utama menampilkan seluruh jurnal yang pernah ditulis dalam tabel
yang dapat diurutkan berdasarkan nomor minggu atau jumlah kata. Setiap
baris menampilkan nomor minggu, status pengumpulan, jumlah kata, dan
tanggal pengiriman.

### Status Pengumpulan
Setiap jurnal memiliki salah satu dari tiga status:
- **Tepat Waktu** — dikirim sebelum batas Minggu 21:00 WIB
- **Terlambat** — dikirim setelah batas waktu
- **Tidak Dikumpulkan** — minggu berlalu tanpa pengiriman

### Penulisan Jurnal Minggu Ini
Tombol "Tulis Jurnal Minggu Ini" membuka editor untuk minggu yang sedang
berjalan. Jika jurnal minggu ini sudah dikirim, tombol ini langsung
membuka tampilan baca.

### Editor dengan Petunjuk Penulisan
Saat menulis, MABA mendapatkan petunjuk pertanyaan reflektif untuk setiap
bagian (What / So / Now). Jumlah kata dihitung secara langsung dan
ditampilkan di bawah editor. Pengiriman memerlukan minimal **300 kata**
dari ketiga bagian gabungan.

### Auto-save Draft
Editor menyimpan draf secara otomatis sehingga tulisan tidak hilang
meskipun halaman ditutup sebelum dikirim.

### Tampilan Baca Per Minggu
MABA dapat mengakses kembali jurnal minggu mana pun yang sudah dikirim
melalui riwayat di tabel. Jurnal yang sudah final tidak dapat diubah.

## Catatan

- Nomor minggu dihitung relatif terhadap tanggal mulai cohort MABA.
- KP dapat melihat jurnal binaannya melalui panel KP (bukan dari halaman ini).
